#  Kickstarter Projects Dashboard

This project is an Exploratory Data Analysis (EDA) of Kickstarter crowdfunding campaigns, built entirely in Power BI. The goal is to understand what factors drive project success, explore trends across categories and countries, and deliver an interactive three-page dashboard.

## 📄 Dashboard Preview
[![Dashboard](./dashboard.png)](./kickstarter-dashboard.pdf)

## 🔍 Key Insights
- Small funding goals have higher success rates
- Music and Film categories perform best
- Shorter durations tend to succeed more
